Types of Data Structures

There are two types of Data Types

  1. Primitive
  2. Non Primitive
    • Linear Data Structures
    • Non-Linear Data Structures
Primitive
Primitive data types are those which are predefined way of storing data by the system. And the set of operations that can be performed on these data are also predefined.
Non Primitive
Non-primitive, or reference data types, are the more sophisticated members of the data type family. They don't store the value, but store a reference to that value.

Examples of Primitive Data Structures are

  • int
  • char
  • boolean
  • float
  • double

There are two types of Non Primitive Data Structures

  1. Linear Data Structures
    1. Stacks
    2. Queues
    3. Linked Lists
  1. Non-Linear Data Structures
    • Trees
    • Heaps
    • Graphs
